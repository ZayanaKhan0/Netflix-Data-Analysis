# Netflix Data Analysis using Python

## Project Overview
This project analyzes Netflix movies and TV shows data using Python libraries like Pandas, Matplotlib, and Seaborn.

The analysis includes:
- Data Cleaning
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Data Visualization
- Insights Generation

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset
Netflix Movies and TV Shows Dataset

---

## Key Analysis Performed
- Movies vs TV Shows comparison
- Top countries contributing content
- Ratings distribution analysis
- Genre analysis
- Release year trends
- Missing values handling
- Duplicate removal

---

## Key Insights
- Netflix contains more movies than TV shows.
- United States contributes the highest content.
- TV-MA is the most common rating.
- Drama and Comedy are dominant genres.
- Most Netflix content was released after 2015.
